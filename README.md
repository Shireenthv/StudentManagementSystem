# Student Management System

A backend-based Student Management System developed using **Java, Spring Boot, and MySQL** to manage student records efficiently. The project provides RESTful APIs for performing CRUD (Create, Read, Update, Delete) operations and follows the MVC architecture for clean and maintainable code structure.

---

## Features

- Add new student records
- View all students
- Update student details
- Delete student records
- RESTful API implementation
- MySQL database integration
- MVC architecture implementation
- Input validation and error handling
